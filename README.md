# dns-print

[![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg)](https://github.com/eczn/dns-print/blob/master/LICENSE) [![npm version](https://img.shields.io/npm/v/dns-print.svg?style=plastic)](https://www.npmjs.com/package/dns-print)

a node cli tools for dns query

![asciicast](./ascii.gif)

``` bash
$ npm i -g dns-print
$ dnsprint google.com  # or dns-print google.com
# with alias:
$ alias dnsp="dns-print"
```

# requiremnet

**node >= 16.0.0**

# -s, --silent

``` bash
$ dns-print -s google.com
```
